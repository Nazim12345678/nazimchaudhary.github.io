Enterprise Data Product Platform

Client: TeamSignal (Security Services) Role: Lead Data Architect Timeline: 2020 - Present

Overview

TeamSignal is a leading security services provider delivering comprehensive solutions across multiple sectors. Built an enterprise-scale data product platform (SOAP) delivering 12 KPIs across four business domains with role-based analytics serving executives, directors, sales teams, and field officers.

The project evolved through two major phases: initial implementation with Azure Data Factory and Databricks (2020), followed by a complete migration to Microsoft Fabric (2024) to leverage modern data platform capabilities.

Business Challenge

TeamSignal operated with fragmented data across multiple systems, preventing unified business intelligence and hindering strategic decision-making. Key challenges included:

  • No standardized KPI framework across business units
  • Manual reporting processes consuming significant resources
  • Lack of role-based data access and visibility
  • Disconnected data sources preventing comprehensive analysis
  • Inability to track performance across Sales, Operations, Admin, and Customer Promise dimensions

Solution Architecture

Phase 1: Azure Data Factory & Databricks (2020-2023)

Implemented medallion architecture (Bronze → Silver → Gold) with the following components:

  • Data Sources: Sage Intacct, EDO systems, HR platforms (PayCor, ADP), BET, and legacy systems
  • Ingestion: Azure Blob Storage landing zone with Azure Data Factory pipelines
  • Bronze Layer: Raw data ingestion with Fabric Lakehouse, distributed copy from various sources
  • Silver Layer: Data transformation using Databricks, quality checks, deduplication, and standardization
  • Gold Layer: Business-ready data warehouse with curated metrics and KPIs
  • Analytics: Power BI dashboards with role-based access control

Phase 2: Microsoft Fabric Migration (2024)

Migrated entire platform to Microsoft Fabric ecosystem for enhanced performance and unified data experience:

  • Fabric Event Streams: Real-time data ingestion capabilities
  • Fabric Lakehouse: Unified Bronze and Silver layers with OneLake integration
  • Fabric Data Warehouse: Gold layer with optimized query performance
  • Fabric Notebooks: Python/Spark transformations replacing Databricks
  • Fabric Pipelines: Enhanced orchestration with better monitoring
  • DevOps Integration: Two-environment CI/CD strategy (Development → Production)

Medallion Architecture: Bronze (Raw) → Silver (Refined) → Gold (Curated)

SOAP Data Product Framework

Developed comprehensive data product delivering 12 KPIs across four business sections:

Sales Section

  • Net Revenue Growth: Year-over-year revenue increase tracking
  • Market Share: Competitive positioning analysis
  • Same Customer Sales: Customer retention revenue metrics

Administration Section

  • Gross Profit: Profitability metrics by service line
  • Days Sales Outstanding (DSO): Cash flow and collection efficiency
  • Accounts Receivable Over 90: Aging analysis and risk management

Operations Section

  • Dedicated Shift: Resource allocation and utilization
  • Completion Patrol: Service delivery performance
  • Completion Labor Efficiency: Productivity optimization

Promise Section

  • Customer Retention: Client loyalty and satisfaction tracking
  • OBX (Officer Engagement): Field officer performance and engagement
  • AFFIRM (CBX): Customer-facing metrics and quality indicators

Role-Based Access Control

Implemented comprehensive security model with tailored views for different organizational roles:

  • Home Office: Full enterprise visibility across all metrics and dimensions
  • Owner: Strategic KPIs with executive-level aggregations
  • Director: Department-level metrics with drill-down capabilities
  • Sales Executive: Sales performance, pipeline, and customer metrics
  • Officer: Individual performance metrics and operational data

All reports embedded directly into TeamSignal's website with seamless single sign-on integration.

Migration to Microsoft Fabric

The 2024 migration to Microsoft Fabric delivered significant platform improvements:

  • Performance: Faster query execution with Fabric Data Warehouse optimization
  • Integration: Unified experience with OneLake eliminating data movement
  • Cost Optimization: Consolidated compute resources and simplified architecture
  • Real-time Capabilities: Event Streams enabling streaming analytics
  • DevOps Maturity: Enhanced CI/CD with two-environment deployment strategy
  • Governance: Improved data lineage and quality monitoring
